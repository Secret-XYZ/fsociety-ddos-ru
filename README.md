# Fsociety DDoS GUI и без графического интерфейса (Termux)
Fsociety DDoS GUI — это приложение для выполнения простых DDoS-атак (распределенных атак типа «отказ в обслуживании») с использованием PyQt5 в качестве графического интерфейса пользователя (GUI) и Requests для отправки HTTP-запросов на целевой объект. Приложение позволяет пользователям настраивать количество запросов, режим скрытности и использование прокси, а также отображает журналы атак в графическом интерфейсе.

# СКРИНШОТ
# Особенности 🚀
· DDoS-атака: многократно отправляет HTTP-запросы на целевой URL 🌐.
· Режим скрытности: использует другой заголовок User-Agent для снижения вероятности обнаружения 🕵️‍♂️.
· Прокси: опция автоматического использования прокси 🌍.
· Журнал атак: отображает статус атаки, включая количество успешных и неудачных запросов, в графическом интерфейсе 📜.
# Предварительные требования ⚙️
Это приложение поддерживает различные операционные системы и требует предварительной установки некоторых зависимостей Python.
**Поддержка операционных систем 💻**
· Windows: Это приложение полностью поддерживает Windows 10/11.
· Linux: Это приложение может работать на дистрибутивах Linux, таких как Ubuntu, Kali Linux и других.
· macOS: Это приложение может работать на macOS с Python 3.x. -- termux (используйте версию без графического интерфейса)
**Необходимые зависимости 📦**
· Python 3.x: Убедитесь, что Python 3 установлен в вашей системе.
· PyQt5: Используется для графического интерфейса.
· Requests: Используется для отправки HTTP-запросов.
# Инструкция по установке 🔧
**На Windows**
  1. Установите Python: Загрузите и установите Python 3.x с [официального сайта Python](https://www.python.org/downloads/) 🐍.
  2. Клонируйте репозиторий: Откройте командную строку и выполните следующую команду:
